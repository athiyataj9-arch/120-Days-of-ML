## 📈 Daily Progress Log

### 📂 Phase 1: Foundations (Days 1–20)

**Day 01: NumPy Foundations**
* **File:** `01_Foundations/day01_numpy.ipynb`
* **Concepts:** Vectorization, Broadcasting, and Array Reshaping.
* **Reflection:** NumPy is the engine of ML. Realized that matrix operations are significantly faster than standard Python loops because of C-extensions.

**Day 02: Pandas & Data Wrangling**
* **File:** `01_Foundations/day02_pandas.ipynb`
* **Concepts:** DataFrame manipulation, Outlier handling with `.mask()`, and Grouped Imputation.
* **Reflection:** Data cleaning is where 80% of an ML Engineer's time is spent. Mastered filling missing values based on category means (e.g., Department-wise salary).
